# <a name="overview-of-azure-deployment-manager"></a>Vue d’ensemble d’Azure Deployment Manager

Deployment Manager vous permet d’utiliser des pratiques de déploiement sécurisé lorsque vous déployez votre service dans de nombreuses régions.

Azure Deployment Manager est en préversion publique. Pour savoir comment vous inscrire à la préversion et utiliser Azure Deployment Manager, consultez [Autoriser des pratiques de déploiement sécurisé avec Azure Deployment Manager](https://docs.microsoft.com/azure/azure-resource-manager/deployment-manager-overview). Pour obtenir un didacticiel, consultez [Utiliser Azure Deployment Manager avec des modèles Resource Manager](https://docs.microsoft.com/azure/azure-resource-manager/deployment-manager-tutorial).
